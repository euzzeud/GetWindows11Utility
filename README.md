# Get Windows 11

## _A utility that allows you to download the ISO of Windows 11._

_This software is an open source [VB.net](https://en.wikipedia.org/wiki/Visual_Basic_.NET) project._

This utility is really easy to use: select the ISO destination folder and press the "*Windows 11's ISO file!*" Button. and you have nothing more to do! (Just wait for the download to finish! üòâ)
